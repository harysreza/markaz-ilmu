# Markaz Ilmu - CMS Website Kajian

Markaz Ilmu adalah website CMS kajian multimedia yang dirancang untuk audiens lintas generasi, memberikan kemudahan akses ilmu syar'i dalam format teks, audio, dan video.

## Tech Stack
- **Frontend**: Next.js (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **Icons**: Lucide React
- **Accessibility**: Customized font scaling & Dark Mode support

## Fitur Utama (Tahap Awal)
- **Sticky Navbar**: Navigasi yang mudah diakses kapan saja.
- **Accessibility Tools**: Pengaturan ukuran font (A-/A+) dan Dark Mode toggle.
- **Hero Bento Grid**: Menampilkan konten utama dan shortcut program secara modern.
- **Multiformat Discovery Feed**: Filter konten berdasarkan tipe media (Teks, Video, Audio).
- **Halaman E-Book & Do'a**: Desain yang ramah lansia dengan font besar dan navigasi simpel.

## Pengembangan Selanjutnya
- Integrasi Strapi CMS.
- Storage Cloudinary (Video/Image) & Uploadthing (Audio/PDF).
- Autentikasi User.

## Cara Menjalankan
1. Clone repository
2. Install dependencies: `npm install`
3. Jalankan server dev: `npm run dev`
4. Buka `http://localhost:3000`
